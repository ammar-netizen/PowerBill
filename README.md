# PowerBill - Electricity Bill Estimator

An Android mobile application for estimating monthly electricity bills. Built with Java and SQLite for offline data storage.

## Features

- Estimate monthly electricity bill based on block rate pricing
- Input field for units used (1 – 1000 kWh)
- Flexible rebate input (0.0% – 5.0%) via slider
- Auto-save calculation results to local SQLite database
- View all saved records in a list view
- View full detail, edit, and delete individual records
- About page with information and app instructions
- Splash screen on app launch
